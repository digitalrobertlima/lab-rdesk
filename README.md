# 🏺 R-Desk Component Lab
Biblioteca de componentes White-label com estética Industrial Organic Luxury.

## 🔑 Auth-Module (Login)
**Estilo:** Dark/Glassmorphism com suporte a CPF, Social e WhatsApp.
**Integração:** 
- Copie a pasta `auth-screen`
- Instale `lucide-react` e `canvas-confetti`
- Use: `<LoginScreen appName="Sua Marca" onLogin={suaFuncao} />`

## 💸 Pix-Engine
**Estilo:** Minimalista com QR Code dinâmico e feedback de confete.
**Integração:**
- Copie a pasta `pix-payment`
- Instale `qrcode.react` e `canvas-confetti`
- Use: `<PixScreen value={50.00} onPaymentComplete={suaFuncao} />`
